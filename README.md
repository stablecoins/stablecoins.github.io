# stablecoins.org

Next.js site for stablecoins.org.

## Requirements

- Node.js 18+
- npm

## Setup

```bash
npm install
```

## Develop

```bash
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
```

## Deploy

Deploys the static export to GitHub Pages:

```bash
npm run deploy
```
